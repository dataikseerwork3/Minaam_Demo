# Minaam Muzaffar - Portfolio

A professional portfolio website for Minaam Muzaffar, an Analytics & Automation Developer at DataIkseer.

## 🎨 Features

- **Responsive Design**: Mobile-first approach with full responsiveness
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Fast Loading**: Optimized CSS and JavaScript
- **SEO Friendly**: Proper semantic HTML structure
- **Easy to Customize**: Well-organized code and comments

## 📋 Sections

1. **Navigation** - Sticky navbar with smooth scrolling
2. **Hero Section** - Eye-catching introduction with call-to-action buttons
3. **About** - Professional summary and key statistics
4. **Skills** - Six main skill categories with icons
5. **Projects** - Featured projects with descriptions and technologies
6. **Experience** - Timeline of professional experience
7. **Contact** - Multiple contact methods
8. **Footer** - Social media links

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with gradients and animations
- **JavaScript** - Interactive features and smooth scrolling
- **Font Awesome** - Icons for skills and contact methods

## 📱 Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: Below 768px

## 🚀 How to Use

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Customize the content with your information
4. Add your social media links in the footer and contact section

## 📝 Customization

### Personal Information
Update the following in `index.html`:
- Name and title
- Contact email
- LinkedIn and GitHub URLs
- Project descriptions
- Experience details

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f5576c;
    /* ... other colors */
}
```

### Projects
Replace the featured projects section with your actual projects, including:
- Project title
- Description
- Technologies used
